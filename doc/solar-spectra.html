As our energy system must urgently transition to a larger dependence on solar energy, I thought it could be worthwhile to walk through the physical limits imposed by the terrestrial solar spectrum.

We start with the reference solar spectrum ASTM G173-03, as defined by ASTM and published by NREL, which is included as ASTMG173 in this package:

dataset <- photoec::ASTMG173

This dataframe is in wide format. To make plotting (with ggplot2) faster, let’s transform it to long format.

# while we are at it, let's give the existing columns more descriptive names
dataset <- dataset %>%
   rename(AM0 = extraterrestrial) %>%
   rename(AM1.5G = globaltilt) %>%
   # direct normal circumsolar
   rename(DNCS = direct.circumsolar)
datalong <- dataset %>% pivot_longer(
   cols = !starts_with("wavelength"),
   names_to = "model",
   values_to = "value") %>%
   mutate(property = "spectral.irradiance")
datalong %>% glimpse()

Now, as a sanity check, let’s compare the spectra returned by this package’s sunlight.ASTM() function with those above (should be an exact match).

They look the same, and indeed, the values (here exemplified by AM1.5G) returned by the dataset and the function are identical, as expected:

identical(
   datalong %>% filter(model == "AM1.5G") %>% filter(property == "spectral.irradiance") %>% pull(value),
   dflong %>% filter(model == "AM1.5G") %>% filter(property == "spectral.irradiance") %>% pull(value))
## [1] TRUE

Whereas the dataset contains only spectral irradiance, the dataframe returned by this package’s sunlight.ASTM() function derives other properties, at the moment absolute and relative cumulative irradiance (irradiance and irradiance.fraction).
